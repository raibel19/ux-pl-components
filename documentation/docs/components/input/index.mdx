# Componente Input

El componente `Input` de UX-PL Components es un campo de entrada de texto altamente versátil y configurable, diseñado para adaptarse a una amplia gama de necesidades de formularios y captura de datos en tus aplicaciones React.

## Características Principales

- **Validaciones Integradas:** Soporte para validaciones numéricas (límites, rangos), longitud máxima, y errores personalizados.
- **Elementos Adicionales:** Permite añadir iconos, texto o botones a la izquierda y/o derecha del campo de entrada.
- **Personalización Avanzada:** Múltiples opciones para sanitizar la entrada, formatear números, y configurar el comportamiento del autocompletado.
- **Estilizado con Tailwind CSS:** Totalmente personalizable usando clases de Tailwind y variables CSS.

### 2. Input Numérico con Límites

Un input de tipo numérico que solo permite valores entre 18 y 99.

<Spacer />
<InputLib
  nativeInputsProps={{ id: 'input-numerico', type: 'number', placeholder: 'Edad' }}
  textLabel="Edad (18-99)"
  showTextLabel={true}
  validations={{
    number: {
      limits: { min: 18, max: 99, showError: true, reset: false },
      between: undefined, // Asegúrate de pasar 'undefined' si no usas una prop opcional anidada
    },
    maxLength: 2,
    showError: true, // showError general para el input
    errorMessages: {
      limitsMin: 'Debes ser mayor de edad.',
      limitsMax: 'La edad no puede ser tan alta.',
    },
  }}
/>
<Spacer />
<DetailsBlock summary="Código">
  ```tsx showLineNumbers title="Input"

    textLabel="Edad (18-99)"
    showTextLabel={true}

        between: undefined, // Asegúrate de pasar 'undefined' si no usas una prop opcional anidada
  ```

### 3. Input con Icono a la Izquierda y Contador

Un input de búsqueda con un icono y un contador de caracteres (si `maxLength` está definido en `validations`).

<Spacer />
<InputLib
  nativeInputsProps={{ id: 'input-icono', placeholder: 'Buscar producto...' }}
  textLabel="Buscar"
  showTextLabel={true}
  leftElement={{
    id: 'search-icon',
    show: true,
    type: 'icon',
    //icon: <Search size={18} />,
    // Asegúrate de que el objeto leftElement esté completo si tiene más props requeridas u opcionales
  }}
  rightElement={{
    // Ejemplo de cómo añadirías el contador si está en rightElement
    elements: undefined, // Si no hay otros elementos
    clear: undefined,
    counter: { show: true, separator: false }, // Mostrar contador
    error: { show: true }, // Suponiendo que quieres mostrar el icono de error si es necesario
  }}
  validations={{
    maxLength: 50, // Para que el contador funcione
    // ...otras validaciones...
  }}
/>
